R言語を用いた実習と座学で，確率論と統計解析の基礎を学ぶためのコースです．

2つの科目があり，それぞれ

  • 数理科学続論 I (Sセメスター 金曜5限) 確率論と統計学を中心とした基礎的な知識を身に付ける
  • 数理科学続論 J (Aセメスター 金曜5限) 多変量解析のさまざまな手法を実データに適用しながら学ぶ

ことを目標としています．どちらもR言語を用いた演習を行うので， R言語に関する部分は一部重複します．
